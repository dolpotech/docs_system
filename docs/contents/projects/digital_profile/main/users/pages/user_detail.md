# Currently Logged In User Details
This document describes the API endpoint that fetches details about currently
logged in user.

## Endpoint:
```http://digital.kobo.local/api/v1/get-user-details```

## Request Method:
`GET`

## Response:
```json
{
    "id": 1, 
    "username": "super_admin", 
    "full_name": " ", 
    "email": "kobo@example.com"
}
```
